# Get metadata information

Retrieves the metadata tracking information data frame from an IST
object.

## Usage

``` r
GetMetaInfo(IST_obj, ...)

# S3 method for class 'IST'
GetMetaInfo(IST_obj = NULL, ...)
```

## Arguments

- IST_obj:

  An IST object

- ...:

  Additional arguments passed to methods

## Value

Data frame containing metadata tracking information

## Examples

``` r
if (FALSE) { # \dontrun{
meta_info <- GetMetaInfo(ist_obj)
} # }
```
